# Chrome Web Store Screenshot Refresh

Use this guide to capture the five raw screenshots for the refreshed BulkStatus Chrome Web Store listing.

Place raw captures here:

`dist/store-listing/raw-captures-0.1.32/`

After the raw captures are available, convert them into final `1280 x 800` marketing screenshots in:

`dist/store-listing/screenshots-marketing/`

The previous store assets were archived here:

`dist/store-listing/archive/2026-06-09-pre-0.1.32-refresh/`

## Capture Settings

- Use the latest unpacked extension from `extension/`.
- Capture at a wide desktop size, preferably `1280 x 800` or larger.
- Keep browser zoom at 100%.
- Use light mode for screenshots 1-4.
- Use dark mode only for screenshot 5.
- Make sure the latest stacked-sheet BulkStatus icon is visible in the app header when possible.
- Avoid sensitive, internal, or confidential URLs in screenshots.
